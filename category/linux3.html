screen bash

screen 直接打开screen

pstree |grep screen

screen -r 恢复到原来的任务

screen -dr 终止连接退回bash　，但不中止程序运行.

screen -r pid 多个screen时恢复到指定的screen

在screen中，CTRL+a ，开一个新的screen窗体。

screen -x 恢复到最近断开的screen

pstree | grep -A2 -B2 screen 查看screen以及上下２行

在一个screen中，CTRL+a 按下n，在多个窗口中切换
